Configuring Atom for Python development

Thu 28 January 2016

Yesterday, out of nowhere, a thought popped up into my head and I migrated from Sublime Text 3 to Atom by GitHub. I happened to play around with Atom for a little while and I have some customizations to share that can make your life easier when working in Python.

Why Atom?

Atom is hackable to the core, open-source and most importantly free. It is built on web technologies. Atom is very customizable with the help of various plug-ins available. Out of the box, it comes with some basic plug-ins to make your life easier.

How to install plug-ins and themes?

All of the packages from the community can be found here. To install, you can use Install tab in settings page. Open settings view (Ctrl + ,) and click on "Install" tab on the left. Search the package name and once results are displayed, click the "Install" button for the desired package and hang on tight.

The process to installing a theme is same as that of package but before searching for a theme, make sure to press the "Theme" toggle on right of the search bar.

Command Line package management

If you'd like to install/manage packages from the command line, Atom provides you option to do that through apm (Atom Package Manager). You can read more about it here.

Packages and Configurations

1. Use spaces instead of tabs

PEP8 styling guide recommends spaces for indentation instead of tabs. You can configure for this by going into Settings menu and scrolling to 'Soft Tabs' and checking it. You should also scroll down and select 'Tab Length' as 4 if it isn't already set so.

2. Linter for Python

A linter is a small program that checks code for stylistic or programming errors. To enable linting support for Python, install linter-flake8 plug-in.

linter-flake8 requires flake8 package to be installed on your system. To do that, you can type in and execute the following command:

pip install flake8

3. Open recent files/folders

Install the open-recent package to enable option to open recent files and folders.

4. Minimap

Minimap is one of the most downloaded plug-ins on Atom. It allows you to have a full preview of your source code like Sublime Text.

5. Highlight selected words

You can highlight a word throughout the file when you double click it with highlight-selected package.

6. Get terminal inside Atom

terminal-plus package allows you to get a Terminal window right inside your text editor.

7. Add ignored files

It can get annoying to have .pyc files show up in the sidebar along with your project files. Just go to 'Settings' page and scroll down to 'Ignored Names' and add *.pyc and the files won't annoy you any longer. You can also add other files to be ignored such as .db etc. or directories such as virtualenv.

8. Enhance the looks of your editor

There are lots of themes to customize the way Atom looks. I use seti-ui and seti-syntax which provides a beautiful dark layout for editing (Screenshots available on the package page).

To use Seti, install the themes, then go to 'Settings' page and click on 'Themes' tab on left side and select 'UI Theme' and 'Syntax Theme' as 'Seti'.

You can also browse for more themes here and pick the one that suits you.
