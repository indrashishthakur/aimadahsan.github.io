about

Hola, I am Aimad Ahsan, a Computer Science student from India. I created this blog with an attempt to write down stuff I learn, and sharing my thoughts. Most of it is related to programming and technology. You can think of this as an effort to give back to the amazing Internet community.

I mostly code in Python (3.x) and JavaScript and I can safely say you're going to see a lot of that here. I also have some experience in C [and PHP too?]. I also like to read but that doesn't go well with procrastination sometimes (I prefer non-fiction over fiction by the way). Apart from that, there isn't really a lot I can talk about myself.

I use GNU/Linux (Arch) operating system with i3 tiling window manager which is currently rocking gruvbox color scheme system-wide (you can find my dotfiles on GitHub). I use Vim and Atom for text editing purposes (depending on the size of project). I'm also trying to learn Git recently.

I guess that's quite a bit about me. If you'd like to ask any question, suggest something or just say hi, you can check out the social media links below or just drop me an email.
